# Tetris
Tetris game written on JavaScript [Meth Meth Method](https://www.youtube.com/watch?v=H2aW5V46khA&t=127s)

# Control

1st PLAYER : <br />
				Q - rotate left, <br />
				E - rotate right, <br />
				D - move right, <br />
				A - move left, <br />
				S - down. <br />
<br />
2nd PLAYER : <br />
				Y - rotate left,<br />
				I - rotate right, <br />
				J - move right, <br />
				G - move left, <br />
				H - down. <br />
