# Tetris
Tetris game written on JavaScript [Meth Meth Method](https://www.youtube.com/watch?v=H2aW5V46khA&t=127s)

#Control

1st PLAYER : 
				Q - rotate left,
				E - rotate right,
				D - move right,
				A - move left,
				S - down.

2nd PLAYER : 
				Y - rotate left,
				I - rotate right,
				J - move right,
				G - move left,
				H - down.
